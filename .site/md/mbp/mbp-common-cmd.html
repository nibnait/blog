# 坑

Updating Homebrew...

  • 法1：

替换brew.git: cd "$(brew --repo)" git remote set-url origin https://mirrors.ustc.edu.cn/brew.git (opens new window)

替换homebrew-core.git: cd "$(brew --repo)/Library/Taps/homebrew/homebrew-core" git remote set-url origin https://mirrors.ustc.edu.cn/homebrew-core.git (opens new window)

  • 法2:

重置brew.git: cd "$(brew --repo)" git remote set-url origin https://github.com/Homebrew/brew.git (opens new window)

重置homebrew-core.git: cd "$(brew --repo)/Library/Taps/homebrew/homebrew-core" git remote set-url origin https://github.com/Homebrew/homebrew-core.git (opens new window)
